# API de tarefas

API REST simples para gerenciamento de tarefas.

## Funcionalidades
- Criar tarefas
- Listar tarefas
- Atualizar tarefas
- Deletar tarefas
- Gerar data de conclusão automaticamente
- Filtrar tarefas atrasadas

## Tecnologias
- Node.js
- Express

## Como rodar
npm install
node index.js
